# Account Branches 

---

## Get Account Branches by Id {{ tag: 'POST', label: 'https://api.voxo.co/branches/{branchId}' }}

<Row>
  <Col>
    Retreive a single branch with specified account id

    ### Parameters

     <strong>Path</strong>

    <Properties>
      <Property name="branchId*" type="Integer" >
        1234
      </Property>
    </Properties>


    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 758,
        "tenantId": 432,
        "name": "Executive"
    }
    ```

  </Col>
</Row>

```bash {{ title: 'cURL' }}
curl --location -g --request GET 'https://api.voxo.com/branches/123' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    [
        {
            "id": 758,
            "tenantId": 432,
            "name": "Executive"
        }
    ```
