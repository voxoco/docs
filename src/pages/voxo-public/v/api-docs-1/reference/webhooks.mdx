# Webhooks 

---

## Get Webhook By Id {{ tag: 'GET', label: 'https://api.voxo.co/webhooks/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve active webhooks by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 23,
        "tenantId": 432,
        "type": "STARTCALL",
        "url": "https://voxo.co",
        "username": "fred",
        "password": "durst",
        "authtoken": "1234"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/webhooks/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```

## Get Webhooks {{ tag: 'GET', label: 'https://api.voxo.co/webhooks?tenantId={tenantId}' }}

<Row>
  <Col>
    <strong>Retrieve active webhooks for authorized accounts.</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="tenantId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    [
        {
            "id": 23,
            "tenantId": 432,
            "type": "STARTCALL",
            "url": "https://voxo.co",
            "username": "fred",
            "password": "durst",
            "authtoken": "1234"
        },
        {
            "id": 24,
            "tenantId": 432,
            "type": "ENDCALL",
            "url": "https://voxo.co",
            "username": "fred",
            "password": "durst",
            "authtoken": "1234"
        }
    ]
    ```
  </Col>
</Row>
