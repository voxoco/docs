
# Phone Numbers (DID)

---

## Get Phone Number By Id {{ tag: 'GET', label: 'https://api.voxo.co/dids/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve phone number (DID) by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 6283,
        "tenantId": 432,
        "number": "6012028038",
        "namePrefix": "",
        "branchId": null,
        "diCommentName": "Grease Collector",
        "allowEmergency": "on",
        "branch": "",
        "tenantName": "The White House",
        "useStatus": "Available"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/dids/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
