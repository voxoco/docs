# Call Queues

---

## Call Queues {{ tag: 'GET', label: 'https://api.voxo.co/queues/{recordId}' }}

<Row>
  <Col>
    Retreive call queue by Id 

    ### Parameters

     <strong>Path</strong>

    <Properties>
      <Property name="recordId*" type="Integer" >
        1234
      </Property>
    </Properties>


    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
        {
            "id": 298,
            "tenantId": 37,
            "name": "Test Queue",
            "number": "",
            "hostServer": "atl-ast2",
            "recording": "no",
            "ringStrategy": "RINGALL"
        }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/queues/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
