
# Call Queue Managers

---

## Call Queue Manager By Id {{ tag: 'GET', label: 'https://api.voxo.co/queue-managers/{recordId}' }}

<Row>
  <Col>
    Retreive queue manager by Id 

    ### Parameters

     <strong>Path</strong>

    <Properties>
      <Property name="recordId*" type="Integer" >
        1234
      </Property>
    </Properties>


    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
        {
            "id": 5,
            "queueId": 298,
            "userId": 1362,
            "userEmail": "fred@gmail.com",
            "timezone": null,
            "userRole": 3,
            "receiveReports": 0
        }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/queue-managers/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
