# Time of Days

---

## Get Time of Day By Id {{ tag: 'GET', label: 'https://api.voxo.co/conditions{recordId}' }}

<Row>
  <Col>
    <strong>Get single time of days record by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" />
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 309,
        "tenantId": 432,
        "name": "test",
        "type": "WEEKTIME",
        "timezone": "America/Chicago",
        "param1": ""
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/conditions/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
