
# Virtual Receptionists

---

## Get IVR By Id {{ tag: 'GET', label: 'https://api.voxo.co/ivrs/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve IVR by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 53,
        "tenantId": 432,
        "name": "main",
        "allowDisa": "on"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/ivrs/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
