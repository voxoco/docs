## Get Feature Code By Id {{ tag: 'GET', label: 'undefined/features/{recordId} baseUrl=' }}

<Row>
  <Col>
    Retrieve Extension by Id

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 47,
        "tenantId": 432,
        "code": "*999",
        "comment": "Play test audio"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/features/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
