
# Voicemails

---

## Get Voicemail By Id {{ tag: 'GET', label: 'https://api.voxo.co/voicemails/{recordId}' }}

<Row>
  <Col>
    <strong>Fetch voicemail by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 5216,
        "fullName": "Grablic",
        "mailbox": "1234",
        "password": "6209",
        "email": "jut@tut.com"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/voicemails/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
