
# Media Files

---

## Get Media File By Id {{ tag: 'GET', label: 'https://api.voxo.co/mediafiles/{recordId}' }}

<Row>
  <Col>
    <strong>Get media file information by Id</strong>

    Retrieving a media file by id will return a url to download the file.

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 1254,
        "name": "billy goat",
        "tenantId": 432
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/mediafiles/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
