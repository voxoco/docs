

# E911 Locations

---

## Get E911 Location By Id {{ tag: 'GET', label: 'https://api.voxo.co/locations/{recordId}' }}

<Row>
  <Col>
    Retrieve single E911 Location by Id

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 6283,
        "diCommentName": "Grease Collector",
        "emergencyNotes": "{\"streetNum\":\"23\",\"streetInfo\":\"Cutlass Pt\",\"location\":\"\",\"city\":\"Hattiesburg\",\"state\":\"MS\",\"postalCode\":\"39402\"}",
        "namePrefix": "",
        "recording": "no",
        "fax": "no",
        "allowEmergency": "on",
        "tenantId": 432,
        "branch": "",
        "department": "",
        "number": "6012028038"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/locations/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
