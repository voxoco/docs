
# Short Numbers

---

## Get Short Number By Id {{ tag: 'GET', label: 'https://api.voxo.co/short-numbers/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve short number by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 1040,
        "tenantId": 432,
        "number": "6543",
        "destnumber": "6011111111",
        "description": "Chicken Brains"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/short-numbers/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
