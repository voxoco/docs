# Call Groups

## Get Call Group By Id {{ tag: 'GET', label: 'https://api.voxo.co/huntlists/{recordId}' }}

<Row>
  <Col>
    Retreive single call group by Id 

    ### Parameters

     <strong>Path</strong>

    <Properties>
      <Property name="recordId*" type="Integer" >
        1234
      </Property>
    </Properties>


    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
        {
            "id": 41,
            "tenantId": 432,
            "name": "main",
            "number": "4321",
            "type": "SEQUENCE"
        }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/huntlists/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
