# Conference Rooms

---

## Get Conference Room By Id {{ tag: 'GET', label: 'https://api.voxo.co/conference-bridges/{recordId}' }}

<Row>
  <Col>
    Retrieve all conference rooms for authorized tenant accounts

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordedId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 39,
        "tenantId": 432,
        "name": "General",
        "number": "1606"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/conference-bridges/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
