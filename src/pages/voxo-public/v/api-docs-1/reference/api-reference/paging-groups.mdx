
# Paging Groups

---

## Get Paging Group By Id {{ tag: 'GET', label: 'https://api.voxo.co/paging-groups/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve paging group by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 37,
        "tenantId": 432,
        "number": "3424",
        "name": "test",
        "twoWayAudio": "yes",
        "meId": 0,
        "checkInUse": "on",
        "pin": 0,
        "manualHeader": ""
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/paging-groups/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
