

# Parking Lots

---

## Get Parking Lot By Id {{ tag: 'GET', label: 'https://api.voxo.co/parkinglots/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve parking lot range by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 421,
        "name": "the",
        "start": 71,
        "end": 74,
        "hosted": "",
        "tenantId": 432
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/parkinglots/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
