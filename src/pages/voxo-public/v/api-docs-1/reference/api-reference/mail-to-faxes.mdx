
# Mail To Fax Users

---

## Get Mail To Fax User By Id {{ tag: 'GET', label: 'https://api.voxo.co/mail-to-faxes/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve single User with Mail to Fax Privileges by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer" >
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 7,
        "tenantId": 432,
        "name": "Freddy",
        "email": "freddy@freddy.com",
        "cidnum": "1111111111"
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/mail-to-faxes/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
