# Call Blocking

---

## Call Blocking By Id {{ tag: 'GET', label: 'https://api.voxo.co/call-blocking/{recordId}' }}

<Row>
  <Col>
    Retreive single call blocking (blacklist) record with Id 

    ### Parameters

     <strong>Path</strong>

    <Properties>
      <Property name="recordId*" type="Integer" >
        1234
      </Property>
    </Properties>


    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
        {
            "id": 620,
            "tenantId": 432,
            "callerId": "3235937002",
            "inserted": "2016-06-10 20:39:59",
            "reason": "car warranty scam"
        }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/call-blocking/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
