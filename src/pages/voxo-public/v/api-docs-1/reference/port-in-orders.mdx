## Get Port In Order By Id {{ tag: 'GET', label: 'https://api.voxo.co/port-in-orders/{orderId}' }}

<Row>
  <Col>

    ### Parameters

     <strong>Path</strong>

    <Properties>

    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
