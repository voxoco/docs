# Phonebooks

---

## Get Phonebook By Id {{ tag: 'GET', label: 'https://api.voxo.co/phonebooks/{recordId}' }}

<Row>
  <Col>
    <strong>Retrieve phone book by Id</strong>

    ### Parameters

     <strong>Path</strong>

    <Properties>
       <Property name="recordId*" type="Integer">
            1234
       </Property>
    </Properties>

    <strong>Header</strong>

    <Properties>
      <Property name="Authorization*" type="String">
        Bearer `{AccessToken}`
      </Property>
      <Property name="Content-Type*" type="String">
        application/json
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    ```json {{ title: 'Response' }}
    {
        "id": 2,
        "name": "OMNIA-1004",
        "includeExt": "no",
        "tenantId": 37
    }
    ```
  </Col>
</Row>


```bash {{ title: 'cURL' }}
curl --location -g \
--request GET 'https://api.voxo.com/phonebooks/1234' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}'
```
