# Firewall IP whitelist

IP address whitelist

It is recommended that you add <strong><em>ALL</em></strong> of VOXO's following IP address ranges and ports on your firewall for real-time traffic. This is important if you have Numbers and devices in different regions and for resiliency purposes (e.g. if North America Atlanta gateways are down, then North America Chicago gateways will be used). VOXO does not guarantee which region the media will egress from since it can depend on which PSTN-SIP Gateway delivers the call to which VOXO region.
