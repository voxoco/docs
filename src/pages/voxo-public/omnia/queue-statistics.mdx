# Queue Statistics

Queue statistics and agent activity in OMNIA

You can find queue statistics by navigating to the 'Return To Admin' link and accessing the 'Queues' section under Reports.

You can:

<ul>
  <li>View and compare queue statistics</li>
  <li>View queue events, and track queue call flows</li>
  <li>View and compare agent activity across multiple queues</li>
  <li>Export data to csv</li>
</ul>
